Cube Draft Setup API

Simple API to setup drafts for our virtual cube drafts

About the API

Welcome! Behind this page magic happens. For a small price of a get-request with the right parameters, python script runs to set up a table for 4 to 9 players and randomises the contents for the packs in less than a second.

After this it sends the whole thing as a json for our host server to manage.

I'm running everything on a free tier so it might not work or be slow at times. If you were to use it, I advice to load this page before as starting up the container might take a handful of seconds.
